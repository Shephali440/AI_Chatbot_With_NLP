This project contains a simple AI chatbot built using Python and NLTK (Natural Language Processing toolkit).
The chatbot can understand basic user inputs like greetings, name inquiries, help, and goodbyes.
It processes text using tokenization and lemmatization and responds based on rule-based logic.

To run:
1. Install requirements: pip install -r requirements.txt
2. Run chatbot: python chatbot.py

Developed using:
- Python 3
- NLTK
